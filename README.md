# One Bit Messenger
PWA clone do Messenger

## Dependencies

- Ruby 2.6.3

- Rails 6.0.1

## Configuration

- Clone the project:
    ```
    git clone git@github.com:gabyrodrigues/one_bit_messenger.git
    ```

- Install the gems:
    ```
    bundle install
    ```

- Run the migrations:
    ```
    rails db:migrate
    ```
